🎯 About Project

Krishna Library Student is a student management and attendance system designed for libraries and study centers.

The project includes:

- Student Registration
- Admin Approval System
- QR Attendance
- WiFi Verification
- Attendance History
- Fee Management
- Push Notifications
- Popup Notices
- Multi Device Login
- Password Reset Requests
- App Update Alerts
